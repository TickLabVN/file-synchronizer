1. Introduction

File Synchronizer (“we”, “us”, “our”) provides file synchronization services via Google Drive. This policy explains how we collect, use, and protect your personal information.

2. Data We Collect

  • Google Authentication Data: OAuth tokens required to access your Google Drive.
  • Application Data: File lists and metadata used for synchronizing.

3. How We Use Your Data

  • To perform file synchronization with Google Drive.
  • To temporarily store metadata for performance optimization.
  • We do not share, sell, or rent your personal data to third parties.

4. Security

We implement industry-standard security measures (HTTPS, encrypted tokens) to protect your data. However, no system is completely secure.

5. Changes to This Policy

We may update this policy over time. Any changes will be posted here, and the “Last updated” date will be revised accordingly.
